核心是，“自己祝福自己”。
脑中“爱”的念头，的善意，就能产生积极的清晰，激发体内生化反应

祝福内容，主要是希望自己好。什么叫自己好？我们不是自私自利，
而是不把追逐与奋斗当作唯一的价值与目标 

```
像是各种兴趣爱好，运动和睡眠，健康和亲密关系，这些幸福的基石不应该被牺牲掉。如果我们把时间和精力投入到这些领域。那么，不需要爬到金字塔尖做一个普通人也可以很幸福，所以正念中的自我关怀不是一种。自私自利的零和游戏，我们不是踩着别人往上爬，搞钱要紧，而是着眼于那些生命中更加根本的维度。

他们能给我们带来长久的幸福



```

对于“愿我安详和快乐”


```
安详不是那种提不起劲儿，心如死灰的状态，而是泰山奔于前而色不变。那种很镇定的感觉，这里的快乐也不是那种啊，

中了彩票的狂喜，而是一觉睡够自然醒。肚子饿了，有饭吃。若无闲事挂心头，便是人间好时节。那是一种生命本身的喜悦。在这里，我们也不是获得安详和快乐，而是保持安详和快乐。所以潜台词并不是我们没有安详和快乐，那只能从外界去寻找，而是他们原本就在我们自己的心里。我们只需要找到他们，培养他们，
```
感觉很难 领悟  感觉自己懂了，但直觉又告诉自己，还得继续悟

```
祝福自己就是不断的敲打我们的内在形象。我是值得被爱的，我是值得获得幸福的，那么我应该也是有很多优点的吧，我应该没那么糟糕吧，我的缺点和做过的坏事应该就是能被原谅的。应该就不算什么了吧，
```
我明白，但“假慈悲”不能彻底解决。我能稍微感受到祝福的力量，但是，没有那种，特别震撼的共鸣


对自己好.... 我感觉很难把握。直接把握字面意思，又容易变成，让自己放纵，想玩的时候玩，想睡的时候睡，对自己好一点。  
但我从内心里厌恶这种，我不知道是因为我对自己要求严格，还是真心觉得这样对自己不好。不过我也知道，对自己苛责，要求自己优秀，这也是对自己不好。  
也许，对自己好是在俩者之间的一种状态，今天的我或者说这几个月的我，比以往任何一段时间的我都接近，因为我经历过“完全放纵”，也经历过“完全苛责压力”，现在的我每天都有一定量的学习，有硬性的任务（冥想、英语、早起、运动），但我都没有觉得痛苦，而是很轻松很开心地迎接各种活动和学习。也许现在就是。“对自己好”？ 但我没感受到逻辑所在。

真正的“对自己好”，可以被定义为：

**一种在“放纵”和“苛责”之间取得的、动态的、智慧的平衡。它既包含了对长期福祉的承诺与自律，又充满了对当下过程的轻松与享受。**
