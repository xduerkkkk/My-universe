
冥想的几个核心：
**1.感受身体**
```
为什么很多现代人没有内在空间呢？一个原因是注意力都放在外界的事物上。很少会去感受自己的身心，另一个原因是比较不敏感，无法感受到内在发生了什么。即使你看了很多文学，哲学，心理学方面的书，自认为是有一个内心世界的，然而那个世界也常常处于思维层面。整天想的都是一些宏大的东西，比如人性，爱情，性别议题，那是关于全人类的内心，但不是你自己的内心。你可以对人类懂得很多，但同时对自己一无所知
```
提高对身心的敏感度，就能探索出一个内在的世界。
不过我现在，没有觉得有一个很明显的内在世界啊，这里在讲的什么意思呢？

**2.专注**


```
练习中，我们选择的锚点也就是专注的对象是呼吸，呼吸最直观的功效是让你慢下来。静下来，这能让你知道自己要做什么，因为慢下来才能三思而后行，静下来很多事情就会自然想明白人生中的重要领悟啊。很多都不是别人告诉你的，

而是自己想通的，而这种想通往往发生在夜深人静的时候或者生病。放假一个人独处的时候，我们能够暂时脱离生活的洪流，冷静的审视自己校准方向，有了目标以后，接下来就是执行。保持从容和冷静的心态，做事情才能有最好的发挥。执行完了，获得了成就，我们还要幸福，那慢下来才能享受生活。静下来就能看到更多的美好，这样目标成就幸福就都有了

```
这里能理解，但可惜并没有在我生活中体现吧，

**3.开放式冥想、标记**
说白了，就是“看清现实”，

```
但是当我们走在街上的时候，如果不是戴着耳机听歌，回着微信，想着自己的事情，而是真的抬头看看街上的人。老人和环卫工人小摊和打折的商铺，你可能就会意识到世界是复杂而多样的。假如成功的标准只有一个。他们都是所谓失败的人，那好像失败也没有那么可怕，他们每个人都找到了自己的活法，你也可能意识到他们都好辛苦的在讨生活，而我至少吃穿不愁，这样一想好像是。失恋就不算什么事儿了，还有可能感觉到大家都好苦，我希望能帮他们做点什么，然后生活就突然有了意义，爱的发电机就开始转动了

```
老师讲的这段话，让我对“慈心”更理解了，是的，每个人都很好。
然后生活是很复杂的，我已经很好了，如果还焦虑未来什么的，那其他人算什么呢？
是可以有追求，但不能苛责自己，说自己失败什么的


```
当然，也会有一些负面经历可能太强烈了，会破坏我们的内心秩序。这个时候，

我们就可以用到标记法来和它保持一点距离，带着一丝轻松的心态去。去经历它这个不是压抑或者否认啊，那样是解决不了问题的，而是一种包容。一种有边界感的接纳，这样我们就能在发泄情绪和压抑情绪之间找到平衡。同时兼容无条件的爱自己和成为更好的自己，这是一个长期冥想，会慢慢解锁的效果。

```
这段话不太能理解，但很期待

**4.慈心**

```
而慈心能够唤起我们心中的爱，然后进一步改善人际关系，让阳光照进我们的心，这不是因为慈心在道德上正确或者正能量。几千年来，所有的冥想大师们在潜入内心的最深处时，找到的都不是恨，而是爱。所以慈心是道法，自然是让我们冷漠的心恢复它原本的样子

```


后面如何冥想？


```
这14天已经在你心里种下了一颗种子。你体验过了什么是安静？知道向内探索是怎么一回事儿？在人生未来的某个阶段，你可能会突然想起来啊，我曾经推开过一扇门。也许现在里面那些东西能帮到我，然后我们就又续上了冥想，是关于生活的。而生活是一辈子的事情，自有它发展的周期，急不来的

```

第一次14天还真的有点痛苦和咬牙，而现在却是带着乐趣，14天完了，到15天没有了还不适应。

挑选适合自己的冥想：

```
第一类感受身体，它侧重解决的问题是身心迟钝，感受缺乏长期紧张，失眠，有各种躯体化症状。它的核心效果用两个关键词概括是感受和放松，那第二类专注呼吸的方法，它侧重解决的是。使心神不宁，缺乏专注，心定不下来，情绪失控，焦虑内耗，

核心效果用两个词来概括是安静。和专注开放式冥想和标记的方法侧重解决的是想的太多，活在自己的世界里。陷入执念，无法自拔，负面感受，无法消化，有冲动，强迫上瘾等等生活失控问题，它核心效果的关键词是。是放下执念，回到当下那辞心的方法，侧重解决的是孤独自闭社恐。自卑愤世嫉俗，不接纳自己，

不会与别人相处，无法维持亲密关系，它的核心效果是。是爱自己爱人类。那以上依然只是理论，最终还是要看你自己练出了什么效果，然后再去匹配自己的需求，

```
