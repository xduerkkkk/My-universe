#### **输入字段的灵活性**​

| 任务类型      | 必需字段                                    | 可选字段             | 数据来源        |
| --------- | --------------------------------------- | ---------------- | ----------- |
| 文本分类      | `input_ids`, `attention_mask`, `labels` | -                | 原始文本 + 标签   |
| 机器翻译      | 同上 + `decoder_input_ids`                | `token_type_ids` | 双语对照文本      |
| 图像分类（ViT） | `pixel_values`, `labels`                | -                | 图片像素 + 标签   |
| 语音识别      | `input_features`, `labels`              | -                | 音频频谱 + 转录文本 |
