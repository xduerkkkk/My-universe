在我做路径规划，初始二维矩阵的时候，我使用了

```python
dp=[[0] for _ in range(3)]*(2)
```


```python
结果为
[[0], [0], [0], [0], [0], [0]]
并不是预想的
[[0, 0, 0], [0, 0, 0]]
```

为啥呢？一个简单的语法搞错了
  **`* 3` → `[0, 0, 0]`**

- **语法**：`[element] * n` 会创建一个列表，其中 `element` 被重复 `n` 次。
 **`[ for _ in range(3)]` → `[, , ]`**

- **语法**：`[expression for _ in range(n)]` 会 **独立执行 `expression` 多次**，每次生成一个新对象。

所以为什么错误句子会得到连续6个【0】，就是连续3个【0】 重复2次嘛

那到底如何初始化二维矩阵
为
```python
[[0, 0, 0], [0, 0, 0]]

显然

dp = [[0]*3 for _ in range(2)]
即可
```
