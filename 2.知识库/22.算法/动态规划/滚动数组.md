
二维数组映射到一维，其实就是“信息残留”，比如当我们初次见到`dp[i]` 时，就是上一行，第i列，留下来的信息。 `dp[i-1]` 是我们见到过，处理过了的东西。这个信息就不是残留，而是我们的新信息，当前行的。
滚动数组还有个关键，就是每次
`for i in range(1, m):` 从第二行开始遍历时
我们要初始化一下 `dp[0]` 
因为二维时，我们是统一初始化行和列
一维我们只能先初始化第一行
每一列的第一个呢，要放在循环里每次处理一下，否则每一行的第一个永远是第一行的第一个


# 爬楼梯

```python
def minCostClimbingStairs(self, cost: List[int]) -> int:
    n = len(cost)
    dp = [0] * (n + 1) # dp[i] 代表爬到第i级台阶的最小花费
    
    # dp[0]和dp[1]都是0，因为可以直接从这两级开始
    
    for i in range(2, n + 1):
        # 要到达第i级，可以从i-1或i-2上来
        dp[i] = min(dp[i-1] + cost[i-1], dp[i-2] + cost[i-2])
        
    return dp[n] # 楼顶在第n级

```

# 机器人走格子

```python
class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[0]*n for _ in range(m)]
        
        # 1. 初始化第一行和第一列
        for i in range(m):
            dp[i][0] = 1
        for j in range(n):
            dp[0][j] = 1
            
        # 2. 遍历其他格子，进行状态转移
        for i in range(1, m):
            for j in range(1, n):
                dp[i][j] = dp[i-1][j] + dp[i][j-1]
                
        return dp[m-1][n-1]

```

# 机器人走格子障碍物版本

``` python
class Solution:
    def uniquePathsWithObstacles(self, obstacleGrid: List[List[int]]) -> int:
        m = len(obstacleGrid)
        n = len(obstacleGrid[0])
        
        # 1. 检查起点是否就是障碍
        if obstacleGrid[0][0] == 1:
            return 0
            
        dp = [[0]*n for _ in range(m)]
        
        # 2. 初始化起点
        dp[0][0] = 1
        
        # 3. 初始化第一列
        for i in range(1, m):
            # 只有在当前格子不是障碍，并且它上面的格子可达的情况下，当前格子才可达
            if obstacleGrid[i][0] == 0 and dp[i-1][0] == 1:
                dp[i][0] = 1
        
        # 4. 初始化第一行
        for j in range(1, n):
            if obstacleGrid[0][j] == 0 and dp[0][j-1] == 1:
                dp[0][j] = 1
                
        # 5. 核心循环
        for i in range(1, m):
            for j in range(1, n):
                if obstacleGrid[i][j] == 1:
                    # 如果当前是障碍，路径数为0
                    dp[i][j] = 0
                else:
                    # 否则，等于左边和上边之和
                    dp[i][j] = dp[i-1][j] + dp[i][j-1]
                    
        return dp[m-1][n-1]
```

```python
class Solution:
    def uniquePathsWithObstacles(self, obstacleGrid: List[List[int]]) -> int:
        m, n = len(obstacleGrid), len(obstacleGrid[0])
        
        # 起点是障碍，直接返回
        if obstacleGrid[0][0] == 1:
            return 0

        dp = [0] * n
        
        # 1. 初始化 dp 数组，模拟第一行的计算
        for j in range(n):
            if obstacleGrid[0][j] == 1:
                break
            dp[j] = 1
        
        # 2. 从第二行开始，滚动计算
        for i in range(1, m):
            # 每一行的第一个格子 (j=0) 需要特殊处理
            # 如果该行的第一个格子是障碍，或者它上面的格子到不了，那么它就到不了
            if obstacleGrid[i][0] == 1 or dp[0] == 0:
                dp[0] = 0
            # else dp[0] 保持为 1
            
            # 遍历该行的其余格子
            for j in range(1, n):
                if obstacleGrid[i][j] == 1:
                    dp[j] = 0 # 遇到障碍，此路不通
                else:
                    # 状态转移
                    # dp[j] (上一行的值) + dp[j-1] (本行左边的值)
                    dp[j] = dp[j] + dp[j-1]
                    
        return dp[n-1]

```
