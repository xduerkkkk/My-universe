# 加减法
两部分收敛稳定，和也一定收敛稳定。这是个直觉
当然我们也可以使用严肃的定理来证明

   
   **我们的目标**：证明 $\lim_{x \to x_0} (f(x)+g(x)) = L_1 + L_2$。用“挑战-应战”游戏来说，就是挑战者给出任意小的误差$\varepsilon$，我们要保证 $|(f(x)+g(x)) - (L_1+L_2)| < \varepsilon$。

*   **第一步：整理战场**
    我们把要控制的目标变形一下：
    $|(f(x)-L_1) + (g(x)-L_2)|$
    根据绝对值的不等式性质（两数之和的绝对值小于等于两数绝对值的和），我们知道：
    $|(f(x)-L_1) + (g(x)-L_2)| \le |f(x)-L_1| + |g(x)-L_2|$

*   **第二步：分摊“误差预算” (The Aha! Moment)**
    现在，我们的任务变成了让两个误差之和 $|f(x)-L_1| + |g(x)-L_2|$ 小于 $\varepsilon$。
    怎么保证这件事呢？最聪明的办法，就是我给你们俩**“分摊预算”**！
    我要求第一个误差 $|f(x)-L_1|$ 小于 $\frac{\varepsilon}{2}$，同时要求第二个误差 $|g(x)-L_2|$ 也小于 $\frac{\varepsilon}{2}$。
    这样，它俩加起来，就必然小于 $\frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$ 了！

*   **第三步：调用已知条件**
    *   因为 $\lim f(x) = L_1$，所以对于“预算” $\frac{\varepsilon}{2}$，我们必然能找到一个“控制范围” $\delta_1$，当 $0<|x-x_0|<\delta_1$ 时，保证 $|f(x)-L_1|<\frac{\varepsilon}{2}$。
    *   同理，因为 $\lim g(x) = L_2$，我们也能找到一个 $\delta_2$，当 $0<|x-x_0|<\delta_2$ 时，保证 $|g(x)-L_2|<\frac{\varepsilon}{2}$。

*   **第四步：整合控制范围**
    如何让两个条件**同时满足**？很简单，我取一个更小的控制范围，即取 $\delta$ 为 $\delta_1$ 和 $\delta_2$ 中那个**较小者**。这样，只要 $x$ 在这个最终的 $\delta$ 范围内，上面两个不等式就必然同时成立。

*   **结论**：我们成功地对于任意的 $\varepsilon$，都找到了一个对应的 $\delta$，完成了证明。这个证明的核心思想就是**“拆分目标，分摊误差”**。
