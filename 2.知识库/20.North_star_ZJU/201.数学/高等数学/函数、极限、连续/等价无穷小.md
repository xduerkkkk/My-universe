# 等价的意义
若 $\lim \frac{\alpha(x)}{\beta(x)} = 1$，则称无穷小 $\alpha(x)$ 与 $\beta(x)$ 等价。
是说他们的比值，极限等于1 
故而是无穷小量无限趋近于0时，缩小的步伐和速率几乎一模一样

# 和差不换
实际上一个函数的精度可以由泰勒展开，展开成多项式来表示。
我们可以解释为，tanx和sinx等价无穷小，因为主部，系数一样，这个决定了速率。
那能不能说x趋于0 tanx - sinx 也趋于0，因为 0 - 0 等于 0
不能，而我们如果做差，tanx -sinx 我们刚好把这个主部减去了，
我们的核心是因为主部一样，tanx比sinx等于1，所以我们才有等价的概念
现在做减法，那么，主部消掉，x引起影响的就是其他更高阶的项了。 而他们的系数是不同的，
更准确地说，相减后**整体结果仍然趋于0**（因为$\frac{1}{2}x^3$在$x\to 0$时也趋于0），但它不再与$x$等价，而是与$\frac{1}{2}x^3$等价。这个差是一个**更高阶的无穷小**。



## “**等价**无穷小”和“**同阶**无穷小”的根本区别
 同阶，速度大致相同，比起其他的比值是0或无穷， 已经好多了，但仍然是速度仍然是有倍数的差距（就是系数的差别），等价就是速度可以说完全一样，因为比值是1
