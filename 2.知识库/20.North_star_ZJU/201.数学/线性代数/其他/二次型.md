**任何一个二次型，都可以被写成 xᵀMx 的形式，但只有当我们“强制”要求 M 是一个对称矩阵 A 时，这种表示法才是唯一的、并且能揭示其几何本质的。**

我们来一步步“发明”这个过程。

---

### **第一步：从二次型到任意矩阵 M (必然性)**

我们还是看那个例子：  
f(x₁, x₂) = 2x₁² + 2x₁x₂ + 2x₂²

我们想把它写成 xᵀMx = $[x₁, x₂] * [[m₁₁, m₁₂], [m₂₁, m₂₂]] * [x₁, x₂]$ 的形式。

我们把右边展开，看看会得到什么：  
$[x₁, x₂] * [m₁₁x₁ + m₁₂x₂, m₂₁x₁ + m₂₂x₂]$  
$= x₁ * (m₁₁x₁ + m₁₂x₂) + x₂ * (m₂₁x₁ + m₂₂x₂)$
$= m₁₁x₁² + m₁₂x₁x₂ + m₂₁x₂x₁ + m₂₂x₂²$

因为 x₁x₂ 和 x₂x₁ 是一样的，我们可以合并它们：  
$= m₁₁x₁² + (m₁₂ + m₂₁)x₁x₂ + m₂₂x₂²$

现在，我们把这个“通用模板”和我们想要的目标进行对比：  
$m₁₁x₁² + (m₁₂ + m₂₁)x₁x₂ + m₂₂x₂² <==> 2x₁² + 2x₁x₂ + 2x₂²$

**对比系数，我们得到了一个方程组：**

1. m₁₁ = 2
    
2. m₂₂ = 2
    
3. m₁₂ + m₂₁ = 2
    

---

### **第二步：从任意矩阵 M 到唯一的对称矩阵 A (选择性)**

请看第三个方程：m₁₂ + m₂₁ = 2。

这个方程有**无穷多组解**！

- 我们可以选择 m₁₂ = 2, m₂₁ = 0。
    
    - 这时 $M = [[2, 2], [0, 2]]$。这是一个**非对称**矩阵。你用它去算 xᵀMx，得到的结果**仍然是**正确的二次型！
        
- 我们可以选择 m₁₂ = 0, m₂₁ = 2。
    
    - 这时 $M = [[2, 0], [2, 2]]$。这也是一个**非对称**矩阵，但它也能用。
        
- 我们还可以选择 m₁₂ = 5, m₂₁ = -3。
    
    - $M =[[2, 5], [-3, 2]]$。它依然能用！
        

**那么，问题来了：在无穷多种可能性中，我们为什么偏偏要选择对称的那一种？**

**数学家的选择：**  
在所有可能性中，最自然、最简洁、性质也最好的一种选择，就是**把交叉项的系数“公平地”分配给 m₁₂ 和 m₂₁**。  
我们规定 m₁₂ = m₂₁。

代入 m₁₂ + m₂₁ = 2，我们得到 2 * m₁₂ = 2，所以 m₁₂ = 1。  
因此，m₂₁ 也必须等于 1。

**这个选择，就为我们创造出了那个唯一的、特殊的、美丽的对称矩阵：**  
$A = [[2, 1], [1, 2]]$







### **第一步：明确我们的“旧世界”和“新世界”**

- **旧世界 (Standard World):**
    
    - **坐标轴：** 标准基 i=[1,0] 和 j=[0,1]。
        
    - **居民的姓名：** 任何一个向量，它的名字叫 x = [x₁, x₂]。
        
    - **物理定律：** 这个世界里“碗”的形状由一个复杂的、带交叉项的公式描述：f(x) = 2x₁² + 2x₁x₂ + 2x₂²。这个交叉项 2x₁x₂ 告诉我们，这个碗是“歪”的，它的主轴没有对齐我们的标准坐标轴。
        
- **新世界 (Eigen-World):**
    
    - **坐标轴：** A 的特征向量 v₁=[1,1] 和 v₂=[1,-1]。这是一个**互相垂直**（正交）的坐标系，但它相对于我们的旧世界是“倾斜”了45度的。
        
    - **居民的新姓名：** 同一个客观存在的向量，在这个新世界里，有了新的名字（坐标），我们叫它 y = [y₁, y₂]。y₁ 是它在 v₁ 方向上的投影长度，y₂ 是它在 v₂ 方向上的投影长度。
        
    - **我们的目标：** 找出这个世界里“碗”的形状的物理定律 g(y) = ?。
        

---

### **第二步：“翻译”——建立新旧世界的联系**

我们怎么从旧名字 x 得到新名字 y？这正是我们驾轻就熟的**基变换**。

我们有“翻译官”矩阵 P，它的列是新世界的基向量：P = [[1, 1], [1, -1]]。

我们知道：  
x_std = P * y_eigen (标准坐标 = 翻译官 * 特征坐标)  
为了方便，我们写成 x = Py。

---

### **第三步：在新世界里重写“物理定律”**

现在，我们把这个“翻译公式” x = Py，代入到我们旧世界的“物理定律” f(x) = xᵀAx 中去，看看会发生什么。

f(x) = (Py)ᵀ A (Py)

利用转置的性质 (AB)ᵀ = BᵀAᵀ：  
= yᵀPᵀ A Py  
= yᵀ (PᵀAP) y

**请看括号里的这一部分：PᵀAP！**  
由于我们的 P 是由**正交的**特征向量组成的（我们甚至可以把它单位化，变成一个真正的正交矩阵），所以 Pᵀ = P⁻¹。

那么，PᵀAP 就等于 P⁻¹AP！  
而我们知道 P⁻¹AP 是什么？它就是那个极其简单的**对角矩阵 D**！  
D = [[3, 0], [0, 1]]

所以，我们那个复杂的二次型，在新世界里的表达式 g(y) 就是：  
g(y) = yᵀDy

---

### **第四步：回答你的问题——它变成了什么样子？**

现在，我们来展开 yᵀDy，看看它到底有多简单。  
g(y) = [y₁, y₂] * [[3, 0], [0, 1]] * [y₁, y₂]  
= [y₁, y₂] * [3y₁, y₂]  
= y₁ * (3y₁) + y₂ * (y₂)  
= 3y₁² + 1y₂²

**这就是最终的答案！**

**“在这个新的、沿着主轴方向的坐标系里，你期望那个复杂的二次型 2x₁² + 2x₁x₂ + 2x₂²，会变成一个什么样子的、更简单的表达式？”**

**答案是：它会变成一个极其纯粹的、只包含平方项的“标准形”：3y₁² + y₂²。**

**这个新表达式告诉我们：**

- **交叉项 x₁x₂ 消失了！** 这意味着，在新坐标系里，这个“碗”是**“摆正”**的。它的主轴完美地对齐了我们的新坐标轴 v₁ 和 v₂。
    
- **新平方项的系数 3 和 1**，不多不少，正好就是我们算出来的**特征值 λ₁ 和 λ₂**！
    

**几何直觉：**  
这个新公式 3y₁² + y₂² 告诉我们：

- 沿着新 x' 轴 (v₁方向)，碗壁的“陡峭程度”是3。
    
- 沿着新 y' 轴 (v₂方向)，碗壁的“陡峭程度”是1。
    
- 因为 3 > 1，所以碗在 v₁ 方向上更“瘦”，在 v₂ 方向上更“胖”。这正是一个椭圆形抛物面
