# 空间变换视角
一个未知向量v，经过我们已知的空间向量变换，传送到了一个已知的位置（等号右边的向量）， 我们的任务就是，找到初始的向量v

`2x + 3y = 7`
`4x + 5y = 13`

这个方程组可以被写成一个矩阵方程：
`[[2, 3], [4, 5]] * [x, y] = [7, 13]`


## 概念一：**解的存在性与唯一性**
*   **唯一解：**
    *   **直觉：** 变换矩阵 `A` 是一个“完美”的变换。它把整个二维平面（输入空间）不重不叠地映射到了整个二维平面（输出空间）。所以，输出平面上的任何一个点（比如 `[7, 13]`），都有一个且仅有一个原始点与之对应。
    *   **考研语言：** `det(A) ≠ 0`，矩阵 `A` **满秩** (`r(A)=n`)，`A` 可逆。

*   **无解：**
    *   **直觉：** 变换矩阵 `A` 是一个“降维”变换。比如，它把整个二维平面“拍扁”成了一条直线。那么，如果目标点 `[7, 13]` **恰好不在这条直线上**，我们当然永远也找不到一个能飞到那里的原始向量。
    *   **考研语言：** `r(A) < r(A|b)` (系数矩阵的秩小于增广矩阵的秩)。

*   **无穷多解：**
    *   **直觉：** 变换矩阵 `A` 同样是一个“降维”变换（拍扁成直线）。但这一次，我们的目标点 `[7, 13]` **恰好就在那条直线上**。那么，一定有一整条直线上的所有原始向量，在被“拍扁”后，都落在了 `[7, 13]` 这个点上。
    *   **考研语言：** `r(A) = r(A|b) < n` (系数矩阵和增广矩阵的秩相等，但小于未知数的个数)。


### 解释

矩阵的秩，是指**整个输入空间**在经过变换后，所形成的**输出空间的维度**。输出空间也叫列空间
所以A满秩，就一定有唯一解，在这个例子中，就是二维---二维。 
A就是系数矩阵，那么增广矩阵呢？ 我们称增广矩阵的秩为r(A|b)，b就是那个等式右边的新向量。
如果`r(A) = r(A|b)`， 意味着，当我们把向量 b 加入到 A 的列向量组里之后，**并没有撑开一个新的维度**。说明向量 b **本来就已经**躺在 A 的列向量们所张成的那个空间里了. 既然目标在射程内，那么解就**必然存在**。至于是唯一解还是无穷多解，就看 r(A) 是不是满秩了。
注意，原始向量v，在一个A矩阵为 3x3 矩阵的例子里，v **永远**生活在三维输入空间 R³ 中

当`r(A) = r(A|b)`时：
- 如果r(A)满秩，则r(A|b)也满秩，那么向量b的维度，和矩阵输入输出空间的维度都相同。 输出空间是将原空间线性变换一下，原空间每一个点都有且仅有一个新对应的点，所以，我们求的原始向量v有唯一解
- 如果r(A)不满秩，则r(A|b)也不满秩。 比如矩阵是3×3的，而秩是2。首先，这俩秩虽不满，但相等，绝对是有一个唯一三维向量经过A矩阵变换后，变换的结果 Av_p 和 b **都是生活在三维空间里的、位于同一个二维平面上的、完全重合的三维向量**。就说空间维度相同，总能找到一个对应的。 这个唯一解叫特解，设**v_p**。 但还有“核空间”的存在。 原来3维空间的某条穿过原点的直线，被压扁到二维时，全部压到了原点。这条直线上的所有向量设为v_n。那么，不仅仅是v_p 变换后能变成b，v_p + v_n这个向量变换后，也能变成b！因为v_n的转换是0.而v_n，一条直线上的向量，有无数个！故而有无穷解


当**r(A) < r(A|b)**：
此时，加入向量b，撑开了新的维度，它的**方向**太特殊了，它指向了一个 A 的列向量们无论如何组合都无法企及的新方向，从而提升了**整个向量组张成的空间**的维度。，比如b向量加入后，让向量们组成的空间变成了3维，比如说r(A)是2， 无论向量v长啥样都只能把他变换成2维度，永远变不成b，所以这种情况无解

**场景：一个“压扁”变换**

*   **世界：** 我们生活在一个标准的三维空间 `R³` 里。有 `i` (`x`轴), `j` (`y`轴), `k` (`z`轴) 三个基向量。
*   **变换矩阵 `A` (一个“投影仪”)：**
    `A = [[1, 0, 0], [0, 1, 0], [0, 0, 0]]`
    这个矩阵 `A` 在干一件非常简单的事：它把任何三维向量的 `z` 分量直接变成 `0`。
    **几何上，`A` 的作用就是把整个三维空间，垂直地“压扁”到 `x-y` 平面上。**

#### **分析这个变换 `A`：**

1.  **输入空间：** `R³`
2.  **输出空间：** 也是 `R³`。这一点很关键！`A` 是一个 3x3 矩阵，它吃进一个三维向量，吐出来的**仍然是一个三维向量**，只不过吐出来的向量，它的 `z` 坐标永远是 `0`。
3.  **列空间 `Col(A)`：**
    *   `A` 的列向量是 `[1,0,0]`, `[0,1,0]`, `[0,0,0]`。
    *   这三个向量能张成什么空间？是整个 `x-y` 平面。
    *   这个 `x-y` 平面，是**内嵌在 `R³` 空间里的一个二维子空间**。就像一张无限大的纸，漂浮在我们的三维世界里。
4.  **秩 `r(A)`：**
    *   因为列空间的维度是2，所以 `r(A) = 2`。这是一个不满秩的变换。



# 总结
矩阵A的**列向量**，是**原空间标准基向量** (i, j, k...) 经过变换后，在**新空间**的位置/坐标。，（原来空间基座标都是单位向量） 代表空间变换的方式  x是我们想研究的对象  b是已知的，是x经过A变换后 的样子。  那我们要研究的，就是到底哪个向量，经过A后，会变成b？ 这个向量我们命名为未知量x。有可能，是唯一解。也就是只能找到一个x。 只是在什么情况，是A这个基向量，张成的空间仍然是原来的空间。 这样的话， A的列向量张成了一个与原空间**维度相同**的新空间，向量肯定是一一对应的。 此时，A满秩。
有可能，是无穷解。  A的基向量，张成的空间比原来空间维度小！ 比如原来是二维平面，A的基向量躺在一条直线上。那原先空间的向量，都会被映射到这条直线上。可想而知，肯定有很多个向量，变成同一个向量。因为这就是压缩嘛。 那如果b恰好就是那条直线上的向量，那我们就是无穷个x。
也有可能无解。同样是降维，跟上个情况的A一样，但此时b，是在直线以外的，这就没办法了，找不到了。

templatize：
假定增广矩阵(A|b) 叫B 。   综上，若A满秩，b无论怎样不会影响事实，  一定唯一解。
若A不满秩，即降维，  此时看下，若B的秩等于A的秩，说明是无穷解的情况。因为b在“那条直线”
若B的秩大于A的秩，说明b在二维平面里，此时无解。
