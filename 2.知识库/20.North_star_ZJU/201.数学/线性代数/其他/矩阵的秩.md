# 初等变换

 - **初等行变换：** 对**行**进行的三种操作（交换、数乘、倍加）。
    
- **初等列变换：** 对**列**进行的三种操作（交换、数乘、倍加）。
 
 为什么我们说“初等变换**不改变**矩阵的秩”？请分别从“行视角”和“列视角”尝试解释。
## 定义
 首先，初等变换应该是说， 交换行的位置，（不能交换列的位置？） 然后在行或列的基础上 用其他行或列 使用系数相加/相减？

 对于行，我们是看待线性方程组的视角。 每一行都是约束条件。那么秩就是**本质上独立的、不冗余的约束条件**”
 那这个条件， 换位置当然不影响判断是否有冗余，然后互相加/减，都是约束条件之间的操作，也是不改变的情况的。
 
 对于列，我们是用空间变换的视角。
 列的加减，不会创造出新的维度。（但只是直觉理解）。 那就是秩不变。
 **初等行变换，虽然改变了列向量本身，但它没有改变列向量之间的“线性关系”**

# 行秩等于列秩
**核空间的维度 = n - 列秩**好理解
列秩是变换后，基向量张成的空间维度（到底精准定义是什么？空间，维度，怎么称呼？），n是原来基向量张成的空间维度，比如说原来是3维，后面变成两维了，那势必是把很多一维的直线压成点了， 所以核空间是一维

行秩是我们，高斯消元后，剩下的不为0的行。
n是最开始的行的个数。

n = 3  行秩 = 1 
我们发现，这个方程组的所有解，都可以被写成这种形式：

# 秩零度定理
**秩-零度定理 (The Main Bridge)**

- 列秩 + 核空间维度 = n (即 r(A) + dim(Null(A)) = n，n是**列数**) 
- **直觉：** “变换后剩下的维度” + “变换中丢失的维度” = “原始输入维度”


# 秩的性质
## **性质一：转置的秩 r(Aᵀ) = r(A)**

**这个性质，就是“行秩 = 列秩”这个核心定理的另一种说法。**

- r(A) 是 A 的**列秩**。
    
- r(Aᵀ) 是 Aᵀ 的**列秩**。
    
- 而 Aᵀ 的列，正好就是 A 的**行**。
    
- 所以，r(Aᵀ) 就是 A 的**行秩**。

因此，r(Aᵀ) = r(A) 这条性质，和“行秩 = 列秩”是**完全等价的**。



 让我们尝试赋予 Aᵀ 这个变换本身一个几何意义。如果 A 是一个从 Rⁿ到Rᵐ的变换，那么 Aᵀ 就是一个从 Rᵐ到Rⁿ的变换。这两个变换在几何上被称为“互为对偶”。它们作用的空间方向相反，但它们“压缩信息”的程度（即有效维度）是完全一样的。所以它们的秩相等。
    

## **性质二：可逆矩阵乘法的秩 r(PAQ) = r(A) (其中 P, Q 可逆)**

- **考研语言：** 矩阵乘以一个可逆矩阵，秩不变。
