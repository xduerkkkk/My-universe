## 补码表示方法
- 法 1 x 是 n 位字长表现，则2 的 n＋1 次方，＋x  即为 x 的补码
   如-1 的补码，
  2 的 n+1 是 1（第·n+2 位·） 0（第·n+1 位） 00000（n 个 0）
   减去 1 是  0                        1                        1 1 1 111

- 法2： 或使用原码所有位（包括符号位）取反再加一


0 的补码原码移码反码？

同样是 n 位，补码表示范围比原码表示范围多 1 个

举例：4 位数， 1000 补码表示 -8！ （此处用法 2，0111+1 是 1000，确实是 8，再带上符号）
而 1000 原码什么也不是！表示-0，但一般忽略。表示 0 时就使用 0000
