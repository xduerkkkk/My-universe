🗂 目录结构

0. 核心 (Core)
  存放战略规划与底层方法论。
  - 00.规划(Plan): 包含北极星目标、阶段性规划、当前推进的项目以及未来的职业探索。
  - 01.学习操作系统My_OS: 我沉淀出的学习理念、心智模型、精力管理系统以及具体的学习方法论。
  - 02.心智急救包: 应对焦虑、情绪波动时的行动指南与自我对话记录。
1. 日常工作流 (Workflow)
  - 10.日记文件: 每日/周/月复盘、番茄钟日志与即时思考。
  - 11.事件收集箱(Inbox): 临时想法与待处理事项的统一入口。
2. 知识库 (Knowledge Base)
  存放经过深度加工的知识。
  - 20.North_star_ZJU: 考研核心科目（英语、数学、408）。
  - 21.AI世界: 机器学习、深度学习、Transformer等AI前沿技术笔记。
  - 22.算法: 数据结构与算法题解。
  - 23.正念: 冥想练习与感悟。
  - 24.工程实践: 环境配置、工具使用、AI prompt等实战经验备忘。
3. 创造物 (Creations)
  我的输出
  - 30.输出文章: 已发布或正在打磨的技术博客。
  - 31.我的项目: 个人开发的开源项目/个人学习项目记录。
4. 不常用 (Archive)
  归档的旧文件、模板以及个人信息备份、个人不成熟思考等。
  大部分包含隐私，不便于公开（个别例外）
