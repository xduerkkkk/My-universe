### **第二阶段：上机作业攻坚（12月13日 —— 12月20日）**

**目标**：用 emu8086 写出那个“5功能菜单程序”。这是大头，分数占比高，**一定要在12.20前搞定**。  
**你的优势**：你有编程思维（C/Python），只需要把逻辑翻译成汇编。

**上机任务拆解（必读）：**  
那个PPT里的任务其实是一个经典的**综合性实验**。

- **工具**：emu8086。这软件极其傻瓜，打开 -> New -> COM template/EXE template -> 写代码 -> Emulate（运行） -> Run。它还能看到寄存器AX, BX的值，非常适合调试。
    
- **难点**：汇编没有 printf 和 scanf。
    
- **破局点（PPT最后几张图）**：**INT 21H 中断**。这是DOS时代的系统调用，相当于OS里的API。
    
    - MOV AH, 09H + INT 21H = 输出字符串 (printf)
        
    - MOV AH, 01H + INT 21H = 输入字符 (getchar)
        
    - MOV AH, 4CH + INT 21H = 程序结束 (return 0)
        

**你的行动清单**：

1. **下载安装 emu8086**。
    
2. **弄懂骨架**：我会给你一个最简单的“Hello World”汇编骨架，你以后所有的代码都往里填。
    
3. **功能实现逻辑（脑子里先有C语言逻辑）**：
    
    - **菜单**：就是一个死循环，打印几行字，读入一个数字，根据数字 CMP 比较，跳转 JE 到不同的函数标签。
        
    - **小写转大写**：ASCII码操作。'a'是97，'A'是65。所以，减去32 (SUB AL, 20H) 就是变大写。
        
    - **找最大值**：设置一个 MAX 变量，遍历数组，CMP 比较，如果你比我大，我就把你存进来。
        
    - **排序**：冒泡排序的双重循环，用 CMP 和 XCHG (交换指令)。
        
    - **显示时间**：直接抄PPT里的 INT 21H (功能号 2CH 取时间)，难点在于把返回的二进制时间转成字符显示在屏幕上。
        

**这部分我会给你提供具体的代码块逻辑，你负责组装和调试。**
