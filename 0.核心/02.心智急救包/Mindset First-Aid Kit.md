# **“创造者”与“考生”思维画像**

创造者是全身心投入、享受知识的学习、总结的。最终，收获的知识也是服务于创造的。
知识是构建体系的材料！哇，一想到能有自己的知识体系就开心！ 面对时间，每一天都是为知识体系添砖加瓦的资源！对于规划，是完全能调整的，因为当视角服务于“创造知识，享受知识”时，没有任何死板的规划，只有不断动态规划的前往目标的路径。 对于失败，就根本没有失败二字！一个方法行不通，说明马上要收获另一个方法论了！
而考生思维，对知识视作“要完成的清单，要刷完的题”，没有乐趣，没有享受。时间，只是去考场这个刑场的刑期， 规划没完成第一反应是笨、不努力。
嗯... 考生思维，应该是小学初中高考，乃至大学期末考试、保研机制，这些固化的机制，以及周围的环境带来的？完全是难以遏制的神经通路。但巧的是，之前我也经历过难忘的伤心事，历经一年半彻底将旧神经通路铲除！所以我也有信心铲除考生思维。
嗯，，我思考过后真的觉得，考生思维百弊无利啊，
需要很努力的，每天不断地进行校准。
首先就是，计划视作对自己的资源最合适地利用，而不是视作必须完成的任务。休息也不能有负罪感，休息是服务于创造者的，灵感、精力的来源。  我是当“对生活片段、类型分体系”时，感觉考生思维冒出来，有一种压力感。但我也不知道为啥？只是当时的感觉，现在看就完全是，服务于自己的，完全没压力啊？ 
所以考生思维还是有点莫名其妙的，我会时刻监控校准的。
